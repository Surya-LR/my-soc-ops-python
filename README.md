## 🎯 What is Soc Ops?

Tired of awkward silences at networking events? **Soc Ops** is a fast-paced bingo game designed to spark genuine conversations at in-person mixers and conferences.

Players get a card filled with fun, icebreaker questions. Find people who match the criteria, get their signatures, and race to get **5 in a row**. Simple. Social. Effective.

Perfect for:
- **Conferences & Dev Days** — Turn attendees into a community
- **Company Mixers** — Break down silos and build team bonds
- **Meetups & Workshops** — Jumpstart networking

---

## 🎮 How It Works

1. **Get Your Card** — Load the game and receive a personalized bingo card
2. **Mingle & Match** — Walk around and find people who match each question
3. **Collect Signatures** — Get their details to verify the match
4. **Get 5 in a Row** — Win by completing a bingo line (horizontal, vertical, or diagonal)
5. **Celebrate** — Leaderboards and prizes for the top players

✅ **Dynamic Question Generation** — Powered by AI, questions adapt to your crowd

✅ **Multiplayer** — Perfect for groups of any size  
✅ **Customizable** — Tailor questions to your event theme

## 🚀 Getting Started

### Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/copilot-dev-days/soc-ops.git
   cd my-soc-ops-python
   ```

2. **Install dependencies**
   ```bash
   pip install -e .
   ```

3. **Run the app**
   ```bash
   soc-ops
   ```

4. **Open in your browser**
   ```
   http://localhost:8000
   ```
